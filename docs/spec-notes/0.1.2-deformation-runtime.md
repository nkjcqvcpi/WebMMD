# WebMMD 0.1.2 Deformation Runtime Spec Notes

This release implements a complete, highly optimized, and deterministic PMX deformation evaluation runtime on CPU (WebAssembly) and GPU (WebGPU).

## 1. Deformation Evaluation Order

The skeletal and morph deformation evaluation stages are strictly organized into sequential phases to guarantee correct dependency propagation:

1. **Initial reset**: Resets manual bone translation and rotation poses.
2. **Morph Evaluation**: Evaluates input morph weights, propagates recursive Group Morphs, evaluates Flip Morphs, and calculates final bone/material morph offsets.
3. **Bone Transform Propagation**:
   - Computes local transform matrix ($M_{local} = T_{manual} \cdot T_{morph} \cdot T_{append} \cdot R_{manual} \cdot R_{morph} \cdot R_{append} \cdot R_{ik}$).
   - Propagates world transforms from parent to child bones in order sorted by transform layer first, then by parent-child relations.
   - Evaluates append translation and rotation grant values.
4. **IK Solving**: Runs the CCD IK Solver iteratively to align links toward the IK target, updating IK rotation cache offsets.
5. **Final Matrix Extraction**: Computes the final world skinning matrices ($M_{skin} = M_{world} \cdot M_{inverse\_bind}$).

## 2. CCD IK Solver

The inverse kinematics solver uses the Cyclic Coordinate Descent (CCD) algorithm:

- Performs up to `loop_count` iterations (capped for performance).
- Constrains link bone rotations within absolute min/max limits.
- Employs robust projection to guard against divide-by-zero or NaN rotation vectors.

## 3. GPU Deform Shader

The WebGPU compute shader is refactored into a single unified pass (`deform.wgsl`):

- **Race-Free Morphing**: Rather than dispatches writing concurrently to the same vertices, each compute shader thread processes a single vertex, accumulating all vertex and UV morph contributions via pre-computed adjacency structures.
- **SDEF skinning**: Rotation center calculation matches the pre-calculated radius origin math in the Saba reference implementation.
- **QDEF skinning**: Implements full Dual-Quaternion blend skinning with proper hemisphere alignment, conjugation, translation reconstruction, and normal rotation.
